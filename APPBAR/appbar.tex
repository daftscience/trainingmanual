\chapter{Introduction}\label{ch:ab_intro}

The Cerner \gls{ab} is used to quickly access Millennium Applications. Each user can set up their \gls{ab}  differently based on their responsibilities with in the laboratory.\marginnote{Like snowflakes, each App-bar is unique.}

This procedure will guide new users through the set up process and demonstrate how the \gls{ab} is used. As you become more familiar with Cerner and find the need to add additional applications, refer back to this guide.


\chapter{Basic Settings}\label{ch:ab_settings}

\paragraph{Launch} the \gls{ab}.\\

\prettyimage{width=\textwidth}{graphics/appbar}

\paragraph{Click} the menu button \appicon{app_bar} to open the \gls{ab} menu and select ``Customize...'' \begin{marginfigure}[8\baselineskip]
    \prettyimage{width=\textwidth}{graphics/appbar_menu}
\end{marginfigure}

\newthought{The ``Customize Cerner \gls{ab}''} window will open with a few options.\\

\prettyimage{width=.8\textwidth}{graphics/customize_menu}

\newthought{Brief description} of these options:
\begin{itemize}
    \small
    \bolditem{Always on top} will keep the \gls{ab} above other windows.
    \bolditem{Allow Floating} allows the \gls{ab} to float anywhere on the screen.
    \bolditem{Large Buttons} increases the size of the \gls{ab} Buttons.
\end{itemize}

\section{Sorting Applications}
It may become useful to organize the applications within the \gls{ab}. This can be done by rearranging the buttons and the addition of \textbf{--Spaces--}

\newthought{The following steps} will walk you through the process of organizing the applications.

\paragraph{Click } the menu button \appicon{app_bar} to open the \gls{ab} menu.

\paragraph{Select} ``Customize...''

\paragraph{Click} the ``Buttons'' tab.\\

\prettyimage{width=\textwidth}{graphics/customize_menu_buttons_added}
\clearpage


\paragraph{Select }Specimen Log-in from the list of applications.\marginnote[2\baselineskip]{\info{The applications selected from the previous section have moved to the top.}}\\

\prettyimage{width=\textwidth}{graphics/spec_login_selected}
\paragraph{Using} the arrows \btn{graphics/move_up} and \btn{graphics/move_down} move \textit{Specimen Log-in} to the top of the list.\\

\prettyimage{width=\textwidth}{graphics/spec_login_top}\marginnote[-8\baselineskip]{\info{As you move the applications on this list, their position will also move on the \gls{ab}}}
\clearpage

\paragraph{Repeat the previous}step until the applications are in the following order:\\

\prettyimage{width=\textwidth}{graphics/organized_app_list}

\section{Grouping the Applications}
\newthought{We can group} certain applications together by adding a \abspace between them.

\paragraph{Select} \GLS{are} and \textit{Click} \btn{graphics/insert_space}\\

\prettyimage{width=\textwidth}{graphics/app_bar_space}\marginnote[-2\baselineskip]{\note{There is now a small space between ``Modify Collections'' and ``Accession Result Entry''}}

\newthought{To make the groupings} more prominent, we can add an additional \abspace.
\paragraph{Select} ``Order Result Viewer" and \textit{Click} \btn{graphics/insert_space}\marginnote[-1\baselineskip]{\info{\gls{are} can't be selected because Cerner won't allow you to add a \abspace next to another \abspace.}}

\paragraph{Select} the \textit{new} \abspace and move it so it's right below the original.\\

\prettyimage{width=.9\textwidth}{graphics/app_list_doublespace}\\

\prettyimage{width=\textwidth}{graphics/app_bar_space}\marginnote[-2\baselineskip]{\note{There is now a small space between ``Modify Collections'' and ``Accession Result Entry''}}


\newthought{The applications} within the \gls{ab} are now organized.

in a configuration that better fits your work flow.\\
