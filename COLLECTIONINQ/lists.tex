
This can be used to view all outstanding \boldcap{Transfer Lists} and \textsc{Collections Lists}.\\

\prettyimage{width=\textwidth}{graphics/colinq_list}

\paragraph{Select} the \boldcap{List} tab.

\paragraph{Select} the \boldcap{List Type} option.\sidenote{This will mostly be used for \textsc{Transfer Lists.}}

\paragraph{Select} the \boldcap{Date}.

\paragraph{Click} \btn{graphics/lists}

\infoblock{For \textsc{Transfer Lists}, it will display all lists which have un-received samples. This means, the samples haven't been \textsc{Logged-in} to a location after being transfered.

Right now, it isn't known how useful the Collection Lists options will be.}
