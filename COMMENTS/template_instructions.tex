Templates are pre-written comments which can be accessed using their mnemonics\sidenote{\refpt{sec:comment_templates} for a complete list of Templates and their mnemonics.}.

Some Templates include blank fields where additional information needs to be entered. These fields will appear as \textsc{Underscores (\_)}.
