Modify Collections is the application used to update the collection details of a container.\\

\prettyimage{width=\textwidth}{graphics/modify_collections}


\newthought{It can be used} to update:
\begin{itemize}
    \item \boldcap{Collection Time}
    \item \boldcap{Collection Date}
    \item \boldcap{Collection ID}
    \item \boldcap{Collection Method}
\end{itemize}

\subsection{Using Modify Collections}

\paragraph{Enter} the \textsc{Accession Number}.

\paragraph{Click} \btn{graphics/retrieve}

\paragraph{Double Click} on the field to modify.

\paragraph{Enter} the new value.

\paragraph{Click} \btn{graphics/shitty_icon} when finished.

\newthought{If the results} have already been verified, Cerner will display a warning.\sidenote{This warning is for situations where the \textsc{Reference Range} is determined, in part, by the time of collection.}\\

\prettyimage{width=.5\textwidth}{graphics/warning}

\paragraph{Click} \btn{graphics/yes}

\newthought{Finally, it will} ask if you'd like some labels.\\

\prettyimage{width=.3\textwidth}{graphics/labels}
