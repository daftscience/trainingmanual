\gls{orv} is often used for canceling orders. The advantage of using \gls{orv} is that orders can be quickly pulled up by scanning the sample's barcode.

If an order status is \textit{\gls{completed}} or \textit{\gls{inprocess}} it cannot be canceled.%
\sidenote{Luckily, \gls{orv} will disable the \btn{graphics/cancel_icon}} If you recall, orders with those statuses already have verified results.%
\sidenote{{\reftable{table:orv_order_status}}.}
