When \gls{pi} is first opened, the \textsc{Select Pending Procedures} window will appear. This window is used to select which assays will be displayed.\\

\prettyimage{width=\textwidth}{graphics/select_procedures}

\newthought{There are three} options which can be selected:

\begin{description}
    \bolditem{Test Site} This is the \textsc{Instrument, Bench,} or \textsc{Section} where the test is performed.
    \bolditem{Procedure} This is a specific test.\sidenote{The \textsc{Procedure} field is Optional.}
    \bolditem{Procedure Status} The status of the tests.\sidenote{Usually, this will be set to \textsc{In-Lab}.}
\end{description}
