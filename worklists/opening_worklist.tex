This chapter will guide you through creating a new \textsc{Worklist} for a batched test. The list will contain all of the \textsc{In-Lab} samples.

Items can be removed from the list, or moved up and down within the list.

\paragraph{Click} the \textsc{New Worklist} \btn{graphics/new_worklist_icon} icon from the tool bar.\\

\prettyimage{width=\textwidth}{graphics/worklist_toolbar}

\paragraph{Select} the type of \textsc{Worklist} from the list.\sidenote{It's short.}\\

\prettyimage{width=.5\textwidth}{graphics/select_worklist}

\paragraph{Click} \btn{graphics/ok}\\

\prettyimage{width=\textwidth}{graphics/worklist_example}


\chapter{Modifying the Worklist}

\section{Moving Items on the List}

\paragraph{Select} the row to move.\\

\prettyimage{width=\textwidth}{graphics/worklist_example}

\paragraph{Click} the \btn{graphics/up} or \btn{graphics/down} arrows.\\

\prettyimage{width=\textwidth}{graphics/list_moved}

\section{Removing an Item}

\paragraph{Select} the item from the worklist.\\


\paragraph{Click} the Remove Row \btn{graphics/remove_row} icon from the tool bar.\\

\prettyimage{width=\textwidth}{graphics/item_removed}

\section{Adding an Item}

This will allow you to add any item which \textsc{qualifies}\sidenote{It needs to be in the \textsc{In-Lab} status and not currently in a worklist.} for the worklist.

\paragraph{Click} the Add Row \btn{graphics/add_item} icon from the tool bar.\\

\prettyimage{width=.6\textwidth}{graphics/add_item_window}

\paragraph{Select} the item to add to the worklist.

\paragraph{Click} \btn{graphics/add_button}

\paragraph{Click} \btn{graphics/close} when finished.\\

\prettyimage{width=\textwidth}{graphics/worklist_example}
